# Unofficial Odin Language Reference
This website is meant to be maximally detailed documentation of the Odin Programming Language. It serves a similar purpose to the [offical Overview](https://odin-lang.org/docs/overview/), but designed to be read non-linearly, as to not overwhelm readers, and to be easy to navigate. 

Get started by navigating via the sidebar. 

This is heavily WIP! Feel free to [contribute](https://github.com/rats159/odin_unofficial_reference)
